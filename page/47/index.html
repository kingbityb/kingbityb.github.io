아이폰 베터리 교체시기는??

최근 아이패드 프로 점검을 받으려고 “대우전자서비스센터”를 찾았다가 팁을 하나 알게되어 공유합니다.

- 아이폰6이상 (이번 신상은 아닙니다 ) 베터리 교체비용 3만4천원입니다.

- 배터리 성능 85%미만이라면 교체 권장

(사실 프로그램에 의한 성능저하라고 이슈가 되어서 진행하는 것이긴하지만요. 저처럼 아직 베터리 교체 안하셨다면 그리고 사용기간이 1뇬이 넘으셨다면 추천입니다.
집에 있는 3대의 아이폰을 12월 말까지 모두 교체할 생각입니다. )

  • 베터리성능 체크하기
    설정>베터리

베터리>베터리 성능 상태

베터리성능 > 성능최대치 확인하기

  • 애플 제품 사용중 문제가 생겼을때 초기에는 통화한번허기 어려웠는데 최근 시도해보니 많이 좋아졌더라구요. 전화연결도 바로 되고 전담 상담사도 배정되구요.
    (여전히 AS는 직영이 아닌 하청형태이긴하지만요. )

    https://getsupport.apple.com

게임 이오스나이트에서 득템!!!

무려 레전드 아이템(펫 렉스)득템.

이게 뭐냐구요?

아 이건 EOS 플렛폼에서 돌아가는 게임 EOS Knights 라는 게임입니다.

  • 방치형 게임 ( 한번 클릭후 일정조건까진 알아서 게임진행, 접속 안되어 있어도 됨)

  • 재료 획득을 통해 아이템을 제작하여 판매할 수 있음

  • ios아이폰에서는 앱스토어에 등록 안됨( 홈페이지에서 다운받고 개발자모드를 이용함)

    https://eosknights.io

  • 다운로드 받으면 경고가 뜸

- 아이폰의 설정 > 일반 > 기기관리 > 신뢰

  • 로그인하려면 EOS 계정이 있어야 함

    Eos Lynx지갑을 통해 만들면 쉬워요

    https://steemit.com/dclick/@kingbit/eos-eos-lynx-1539764867447
